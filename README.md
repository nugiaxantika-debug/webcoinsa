# TikTok Top Up Coin Website

🎮 Website untuk top up coin TikTok dengan design yang sama seperti referensi.

## ✨ Fitur

- 💳 8 pilihan paket coin + custom
- 👤 Input username
- 💰 Harga otomatis update
- 🛒 Order summary modal
- 💳 Pilihan metode pembayaran (Card & PayPal)
- ⏱️ Countdown timer 60 detik
- ✅ Success notification
- 📱 Responsive design

## 🚀 Deploy ke Vercel

Pilih salah satu cara:

### 1. GitHub Integration (Paling Mudah)
```
1. Buka https://vercel.com
2. Klik "Add New Project"
3. Pilih repository "webcoinsa"
4. Selesai!
```

### 2. Drag & Drop
```
1. Buka https://vercel.com/new
2. Drag folder ke area upload
3. Selesai!
```

### 3. Vercel CLI
```bash
npm i -g vercel
vercel
```

## 📁 File Structure

- `index.html` - HTML markup
- `style.css` - Styling & responsive
- `app.js` - JavaScript logic

## 🎯 Cara Kerja

1. Input username → 2. Pilih paket → 3. Klik Recharge → 4. Modal order → 5. Select payment → 6. Pay now → 7. Processing → 8. Success

## 💻 Tech Stack

- CSS3 (responsive)

### Menambah paket
